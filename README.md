# DOPS1-LeadPencil
This is the first Daily Observational Problem Solution - trying to solve the short lead wastage problem.

This is the problem I go through on a daily basis. I used a pencil, and mostly a mechanical pencil for homework and even doodling. I am very cautious person like that. I prefer using a pencil over a pen as it helps me be more creative without being worried about the mistakes I make. But the issue that pains me always is throwing away that last piece of the lead, which I buy with my below bare minimum pocket money. Why should I throw away the commodity I buy, for which I pay in full? Instead of looking at the financial point of view of this problem, I wanted to see if I can find some solution from the technological point of view. What if I can somehow either bind the broken end pieces of two or more leads? 

I actually have to talk about the chocolate wrapper problem here, which actually gave me this idea. Fundamentally, the chocolate wrapper question asks you how many chocolates can you get if you have some sum of the money, say Rs.20. If each chocolate is worth Rs.4, and the shopkeeper give a chocolate for returning 3 wrappers, how many chocolates can you get? Here, the solution will be 7, since you buy 5 chocolates with the initial amount, you get 5 wrappers. Giving 3 wrappers gets you two chocolates. Again, you get 1 wrapper, and with the remaining 2 wrappers and this new one, which again results in another chocolate!

Similarly, why can we use the multiple used leads and join them together to create a new lead?
